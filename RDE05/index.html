Pagina Web e Website

Pagina Web

É um arquivo onde é possivel inserir textos, imagens, tabelas, planilhas etc.. Coisas que permitem a navegação de uma pagina ou sesão pra outra

Websites

São uma coleção de paginas da web armazenadas em uma pasta ou subpasta que estão relacionadas com um servidor..

Arquitetura Cliente-Servidor

Client-Side

Tecnologias que são capazes de atender ao cliente como HTML, CSS, JS

Server-Side

Tecnologias que são capazes de atender ao cliente como Java, PHP, MySql...

Sintese da aula de hiperlinks

Hiperlink ou link

Trata-se de uma referencia que está dentro de um documento em hipertexto que direciona para outras partes do site ou de outros sites, havendo tipos diferentes como

  1. Remotos
  2. Internos
  3. Locais
  4. Imagens
  5. Com indereço de correio
  6. Com indereço de arquivos

Sua marcação:

Acontece se utilizando a tag a e a propriedade href, responsavel por fazer a referencia para o conteúdo desejado..

Sintaxe


Imagem dos slides do professor Francis... Todo o crédito a ele..

Hiperlinks Remotos

São os links que fazem referencia a sites e paginas da web, sendo assim, em href, colocamos a URL do site

Temos duas propriedades legais para links remotos, a _blank que abre o site em outra guia da web e a _self que abre na mesma guia, as duas sao utilizadas na opção target

Hiperlinks Internos

São os links que fazem referencia a outra parte do mesmo site, podendo acessar outros paragrafos e seções..

Hiperlinks Locais

São os links que fazem referencia a outras paginas do mesmo site, tendo que colocar o nome do arquivo html dentro da tag href..

Hiperlinks em imagens

São os links que fazem referencia a outras paginas, juntando com a tag de imagem

Hiperlinks com indereço de correio

São os links que fazem referencia a um indereço eletronico, um exemplo esta disponivel ai embaixo direcionando para o meu email

Hiperlinks com indereço de arquivos

São os links que fazem referencia a arquivos e podem fazer download dos mesmos, com as tags download e type
